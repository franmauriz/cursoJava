Frases HTML

EM: se utiliza para dar significado a una parte del texto

STRONG: Se utiliza para resaltar un elemento del texto

Q: Se utiliza para poner comillas

CODE: Se utiliza para poner el texto como si fuera un lenguaje de programacion

ABBR: Se utiliza para poner abrevaciones

ACRONYM: se utiliza para poner acrónimos

DFN: Se utiliza para poner difiniciones

KBD: se utiliza para que el usuario sepa que tiene que introducir texto

VAR: Se utiliza para saber que es una variable
